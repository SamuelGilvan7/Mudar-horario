# ⏰ Hora do Dia

Este é um projeto simples feito com **HTML, CSS e JavaScript**, que muda a interface de acordo com a hora atual do sistema.

---

## 📌 Funcionalidades

- Mostra a hora atual
- Exibe uma mensagem dinâmica
- Muda a imagem conforme o período do dia:
  - 🌅 Manhã
  - 🌞 Tarde
  - 🌙 Noite
- Altera a cor de fundo automaticamente

---

## 💻 Tecnologias usadas

- HTML5
- CSS3
- JavaScript (DOM + Date)
